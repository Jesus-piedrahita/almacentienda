# AI Agent Skills

## Available Skills

### Generic Skills

| Skill | Description | When to Use | Source |
|-------|-------------|-------------|--------|
| `typescript` | Const types, flat interfaces, utility types | Writing TypeScript types/interfaces | local |
| `react-19` | React 19 patterns, React Compiler | Writing React components | local |
| `zustand-5` | Zustand 5 state management patterns | Using Zustand stores | local |
| `axios` | HTTP client patterns and best practices | Making HTTP requests with axios | local |
| `frontend-design` | Production-grade frontend interfaces with distinctive design | Building web components, pages, or applications | github |
| `frontend-design-system` | UI designs with design tokens, layout rules, motion guidance | Creating consistent design systems | github |
| `frontend-ui-animator` | UI animations for Next.js + Tailwind + React | Adding animations or motion to UI | github |

### Meta Skills

| Skill | Description | When to Use | Source |
|-------|-------------|-------------|--------|
| `skill-creator` | Create new AI agent skills | Creating new skills | local |
| `skill-sync` | Sync skill metadata to AGENTS.md | After creating/modifying a skill | local |

## How to Use

Load a skill when you start working on related tasks:

```
Load skill: {skill-name}
```

Example:
```
Load skill: react-19
Load skill: typescript
```
